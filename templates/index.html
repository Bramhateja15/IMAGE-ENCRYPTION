    AN IMAGE ENCRYPTION ALGORITHM BY USING SECURE KEY GENERATION TECHNIQUES

    To enhance security and authentication, this system integrates Elliptic Curve Cryptography (ECC) for image encryption and Local Binary Pattern Histogram (LBPH) for live face authentication during both registration and login. ECC offers high security with smaller key sizes, making it ideal for resource-constrained environments by ensuring encrypted visual data remains confidential with minimal computational overhead compared to traditional methods like AES. Meanwhile, LBPH provides robust real-time face recognition, resilient to changes in lighting and facial expressions, ensuring reliable biometric authentication. The combination of ECC and LBPH strengthens user access control by safeguarding data confidentiality and enabling seamless, secure authentication. Performance evaluations demonstrate notable improvements in speed, accuracy, and efficiency, making the system well-suited for real-time security applications.
